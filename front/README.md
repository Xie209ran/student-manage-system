## 验收步骤

### 1. 打开浏览器测试
```
直接在浏览器中打开 front/index.html 文件
```

### 2. 测试登录功能
- 访问页面，应该看到登录界面
- 不输入任何内容点击登录 → 显示"请输入用户名"错误
- 输入用户名但不输入密码 → 显示"请输入密码"错误
- 输入密码少于6位 → 显示"密码长度不少于6位"错误
- 输入 admin / 123456 → 显示"登录成功"并跳转到主页

### 3. 测试主页面布局
- 登录后应该看到完整的主页面布局
- 左侧是深色侧边栏菜单
- 顶部是白色导航栏，显示"仪表盘"标题
- 右上角显示"管理员"和用户头像
- 中间是内容区域

### 4. 测试菜单切换
- 点击不同的菜单项，顶部标题应该相应变化
- 当前选中的菜单项应该高亮显示
- 内容区域显示对应的占位文字（如"仪表盘页面 - 开发中"）

### 5. 测试退出登录
- 点击右上角用户信息
- 选择"退出登录"
- 弹出确认对话框
- 点击确定后返回登录页面

### 6. 检查控制台
- 打开浏览器开发者工具（F12）
- Console标签应该没有JavaScript错误
- Network标签可以看到加载的CDN资源

### 7. 测试响应式
- 缩小浏览器窗口宽度到768px以下
- 侧边栏应该自动折叠，只显示图标
- Logo文字应该隐藏

## 技术要点

1. **Vue 3 Composition API** - 使用setup语法
2. **Element Plus组件** - el-container, el-menu, el-form等
3. **本地存储** - localStorage保存token和用户信息
4. **组件化** - 每个页面都是独立组件
5. **条件渲染** - v-if控制登录页和主页面切换
6. **事件处理** - @select, @command等事件绑定
