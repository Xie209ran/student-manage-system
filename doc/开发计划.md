# 教学数字平台后台管理系统 - 开发计划

## 说明

- 本计划按执行顺序排列，前面的任务必须先完成
- 每个任务是小颗粒度的，一次对话可以完成
- 任务之间尽量无依赖，如有依赖已在前置任务中完成
- 开发顺序：前端页面（假数据）→ 后端接口 → 前后端对接

---

## 第一阶段：项目基础搭建（1天）

### 任务1：创建数据库和初始化表结构
**任务内容**：根据数据库设计文档，创建MySQL数据库和8张表

**涉及文件**：
- `doc/数据库设计.md`（参考）
- MySQL数据库

**前置任务**：无

**【验收点】**
1. 数据库edum创建成功
2. 8张表全部创建完成（sys_user、class、student、course_schedule、attendance、score、homework、homework_submission）
3. 所有表的字段、类型、注释正确
4. 所有索引创建成功
5. 字符集为utf8mb4，引擎为InnoDB

**【验收标准】**
- 执行 `SHOW TABLES;` 能看到8张表
- 执行 `DESC sys_user;` 等命令能看到正确的字段结构
- 插入测试数据无错误
